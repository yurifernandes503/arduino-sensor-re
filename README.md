# Sistema de Assistência de Ré com Arduino

Projeto acadêmico desenvolvido para a disciplina de Microcontroladores da UNIRUY/Wyden.

## Objetivo

Desenvolver um sistema capaz de auxiliar manobras de ré através da detecção de obstáculos utilizando um sensor ultrassônico HC-SR04.

O sistema informa a proximidade de objetos por meio de LEDs e alertas sonoros emitidos por um buzzer.

## Componentes Utilizados

* Arduino Uno
* Sensor Ultrassônico HC-SR04
* Protoboard
* LEDs
* Resistores
* Buzzer
* Jumpers

## Funcionamento

O sensor HC-SR04 mede continuamente a distância entre o sistema e possíveis obstáculos.

Conforme a distância diminui:

* LED Verde → distância segura
* LED Amarelo → atenção
* LED Vermelho → obstáculo próximo
* Buzzer → alerta sonoro progressivo
